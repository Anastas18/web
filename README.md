# Task An Manager

**Task An Manager** — це вебзастосунок для ефективного керування особистими завданнями, планування часу та організації щоденної діяльності.  
Проєкт розроблено як **особистий менеджер завдань**, що поєднує простоту інтерфейсу, адаптивний дизайн і зручність використання на будь-якому пристрої.

---

## Основні можливості

-  **Створення завдань** — додавання, редагування та видалення записів;
-  **Календар** — перегляд і планування задач за датами;
-  **Категорії** — сортування завдань за тематичними групами;
-  **Пошук** — швидке знаходження потрібних задач;
-  **Профіль користувача** — персоналізований доступ до функцій і налаштувань.

---

## Мета проєкту

Метою розробки є створення зручної інформаційної системи, що допомагає користувачу ефективно організувати свій день, контролювати дедлайни та досягати поставлених цілей.  
Застосунок реалізовано в рамках навчальної дисципліни **«Інтернет-технології та проєктування WEB-застосувань»**.

---

## Використані технології

- **HTML5** – розмітка сторінок;  
- **CSS3 (Flexbox, Grid)** – адаптивна верстка та стилізація;  
- **JavaScript (ES6)** – реалізація інтерактивних елементів і логіки роботи інтерфейсу;  
- **Git / GitHub** – система контролю версій і зберігання коду.
